# Todo List App – RevoU Project

A modern, minimalist, dark-mode Todo List App built using **HTML, CSS (Tailwind + custom styles), and Vanilla JavaScript**.  
This project was created as part of a RevoU mini project assignment.

---

## Features

### Core Features
- Add new todos (title required, description optional, due date required)  
- Delete todos  
- Mark todos as completed  
- Automatic status categorization:
  - Today  
  - Upcoming  
  - Expired  
  - Completed  
- Filter todos using a dropdown  
- Input validation for required fields  

### UI Features
- Fully responsive dark mode  
- Smooth fade-in animations  
- Minimalist glowing accents  
- Clean, modern cards  
- Tailwind CSS + Font Awesome icons  

---

## Technologies Used

- **HTML5**  
- **Tailwind CSS (CDN)**  
- **Custom CSS**  
- **Vanilla JavaScript**  
- **Font Awesome Icons**

## Notes

- **This app does not use a database** – all data is stored in memory and will be lost when the page reloads.  
- Designed to be clean, readable, and easy to extend.
